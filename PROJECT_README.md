# 小区家园通 - 社区工单管理系统

## 项目概述
基于Vant UI + Vue 3 + TypeScript开发的移动端社区工单管理系统，支持业主报修投诉、管理员审核、物业处理等完整业务流程。

## 技术栈
- **前端框架**: Vue 3 + TypeScript
- **UI组件库**: Vant (移动端专用)
- **状态管理**: Pinia
- **路由管理**: Vue Router
- **构建工具**: Vite
- **样式**: 移动端适配，响应式设计

## 功能特性

### 1. 公共登录页面
- 角色选择：业主/管理员/物业
- 业主登录：6位数字口令（示例：123456）
- 管理员/物业登录：账号密码登录
- 忘记密码功能

### 2. 业主端功能
- **首页**: 欢迎信息、功能入口、工单列表、消息通知
- **报修**: 报修类型选择、楼栋信息、问题描述、图片上传、联系电话
- **投诉**: 投诉类型选择、投诉内容、相关证据上传
- **工单详情**: 完整工单信息、处理进度、处理结果查看

### 3. 管理员端功能
- **首页**: 数据统计概览（待审核、已审核、已处理）
- **待审核列表**: 搜索筛选、批量审核、工单状态查看
- **工单审核**: 详细信息查看、批准/驳回操作、审核意见填写

### 4. 物业端功能
- **首页**: 待处理统计、已处理统计
- **待处理列表**: 管理员审核通过的工单列表
- **工单处理**: 处理说明填写、处理图片上传、处理人员登记

## 核心流程

### 工单流转流程
1. **业主提交** → 管理员审核 → 物业处理 → 业主查看结果
2. **状态管理**: 待审核 → 审核通过/驳回 → 处理中 → 已完成

### 权限控制
- 业主：仅能查看和操作自己的工单
- 管理员：只能审核，不能处理工单
- 物业：只能处理已通过审核的工单

## 数据模型

### 工单数据结构
```typescript
interface WorkOrder {
  id: string                    // 工单编号
  type: 'repair' | 'complaint' // 工单类型
  subtype: string               // 具体分类
  building: string              // 所在楼栋
  description: string           // 问题描述
  phone: string                 // 联系电话
  images: string[]              // 相关图片
  status: string                // 工单状态
  submitTime: string            // 提交时间
  ownerName: string             // 业主姓名
  reviewTime?: string           // 审核时间
  reviewer?: string             // 审核人
  reviewComment?: string        // 审核意见
  handleTime?: string           // 处理时间
  handler?: string              // 处理人员
  handleDescription?: string    // 处理说明
  handleImages?: string[]       // 处理图片
}
```

## 开发说明

### 环境要求
- Node.js 14+ (推荐使用最新LTS版本)
- npm 或 yarn 包管理器

### 安装运行
```bash
# 安装依赖
npm install

# 开发环境运行
npm run dev

# 生产环境构建
npm run build
```

### 项目特点
1. **完整的移动端适配**: 使用Vant UI组件库，完美适配各种手机屏幕
2. **TypeScript支持**: 完整的类型定义，提高代码质量
3. **模块化架构**: 清晰的代码结构，易于维护和扩展
